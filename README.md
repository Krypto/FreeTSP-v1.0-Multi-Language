BitTorrent Tracker Source

<b>Requirements</b>

1: PHP 5.3 > Above

2: Apache Web Server 2 > Above or NGINX
<br>
3: MySql Server
<br>
4: Memcached Server

5: Memcache Extension
<br>
6: MySqli Extension (if not exists)
<br>
7: mbstring
<br>
8: gd2
<br>
9: zlib
<br>
10: APC (optional)
